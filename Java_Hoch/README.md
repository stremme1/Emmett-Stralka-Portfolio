## 🚀 Getting Started
1. **Prerequisites**
   - JDK 11 or higher
   - Maven or Gradle
   - IDE (IntelliJ IDEA recommended)

2. **Build and Run**
   ```bash
   # Using Maven
   mvn clean install
   mvn exec:java -Dexec.mainClass="com.example.Main"

   # Using Gradle
   ./gradlew build
   ./gradlew run
   ```
